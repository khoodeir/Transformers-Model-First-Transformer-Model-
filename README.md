# Transformer Sentiment Analysis - First Model

This repository contains my **first Transformer model** built for a simple **sentiment analysis** task using **TensorFlow** and **Keras**.

The main purpose of this project is to learn the fundamentals of the **Transformer architecture** and understand how attention-based models work in Natural Language Processing (NLP).

## Overview

In this project, I implemented a basic Transformer-inspired model that can classify short text sentences into:

- **Positive**
- **Negative**

This is an educational project created as part of my Deep Learning learning journey.

## Features

- Text preprocessing
- Tokenization and sequence padding
- Embedding layer
- Multi-head self-attention
- Layer normalization
- Feed-forward neural network
- Sentiment classification output

## Tech Stack

- Python
- TensorFlow / Keras
- NumPy

## Why This Project?

This project is important to me because it is my **first hands-on implementation of a Transformer model**.

It helped me move from traditional sequence models like **RNN** and **LSTM** to modern attention-based architectures.

## Learning Goals

By building this model, I aimed to understand:

- How text is converted into numerical form
- How embeddings represent words
- How self-attention works
- How Transformer layers are structured
- How to train a basic NLP classification model

## Limitations

Since this is a beginner project:

- The dataset is very small
- The model is simple
- The focus is on learning, not performance

## Future Work

Possible next steps include:
